# Suggested Commands
- List files in PowerShell: `Get-ChildItem -Force`
- Search text fast: `rg -n "pattern" .`
- Search files: `rg --files`
- Open the site locally via simple HTTP server from repo root: `python -m http.server 8000`
- Then browse: `http://127.0.0.1:8000/index.html`
- Quick check current Codex config: `Get-Content $HOME\.codex\config.toml`
- Verify Playwright MCP package resolves: `npx -y @playwright/mcp --help`
- Git status: `git status --short`
