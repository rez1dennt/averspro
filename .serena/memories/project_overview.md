# Project Overview
- Purpose: static Russian-language marketing website for the company "АверсПро" offering 1C consulting/support services.
- Tech stack: plain HTML, CSS, and vanilla JavaScript; no package manager, build system, or backend detected in the repository root.
- Structure: multiple page-specific HTML files in the root (`index.html`, `contacts.html`, `projects.html`, service pages, `blog.html`), shared styles in `style.css`, shared behavior in `script.js`, images in `images/`, local helper dirs `.claude`, `.playwright-mcp`, `.serena`.
- Runtime model: open the HTML files directly or serve the root as a static site from a local HTTP server for browser testing.
- Environment: Windows / PowerShell.
